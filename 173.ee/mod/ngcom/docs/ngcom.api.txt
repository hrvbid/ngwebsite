	Assign values to variables in the tao.vshr pool
	S	vshrSetNode( instance, array(var=>value [,...]) );
		instance	ffu
		var 		the name of the js var
		value		the value to assign to the var
		
	Accessing variables from the tao.vshr pool
	C	tao.vshr.var.dat
	
		ngCom self assigns httpscr and httpurl by
		ngACo::vshrSetNode('ngcomvshr',array('httpsrc'=>PHPWS_SOURCE_HTTP, 'httpurl'=>PHPWS_HOME_HTTP));
		and the variables are accessible as
		tao.vshr.httpsrc.dat and tao.vshr.httpurl.dat
		(note: classical known and used as the source_http variable)
		
	Assign variables with values to the tao vpool
	S	vpoolSetNode( instance, array(var=>value [,...]), [jar]) );
		module		is generated automatically, if specified is ignored
		instance	a unique id
		var 		the name of the js var
		value		the value to assign to the var
		jar			an array, passed to the vpool json encoded
	
	Accessing variables from the tao.vpool
	C	tao.vpool.module.instance.varname
		tao.vpool.module.instance.mod	is the module value
		tao.vpool.module.instance.iid	is the instance value
		tao.vpool.module.instance.$jso	is the json encoded string
		tao.vpool.module.instance.$jso.toArray()	maps the jso data to an array
